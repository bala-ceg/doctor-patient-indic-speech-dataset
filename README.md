# Doctor–Patient Indic Speech Dataset

Multilingual doctor–patient conversational speech dataset.

## Languages
Tamil, Hindi, Telugu, Malayalam, Kannada, Marathi, Gujarati, Bengali, Punjabi

## Structure
audio/<language>/convo_001.mp3  
transcripts/<language>/convo_001.txt  
transcripts/english/convo_001.txt  

## Scenario
Natural conversational doctor–patient clinic dialogue.

## Speakers
2 (Doctor + Patient)
